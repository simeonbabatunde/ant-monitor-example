Class PacketProcessor

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.logging.PacketProcessor

  • public class PacketProcessor
extends java.lang.Object
    This class contains various methods for logging, mapping, and parsing packets. These methods are typically called by the basic PacketConsumer to hide the complexity of the implementation from children of PacketConsumer.
